# 🍳 ChefMate AI

**ChefMate AI** is an AI-powered cooking learning assistant designed to help beginners learn cooking through interactive lessons, real-life examples, quizzes, personalized feedback, and AI-powered question answering.

Built using **Streamlit** and **Google Gemini AI**, ChefMate AI acts as a friendly virtual cooking tutor that makes learning simple, engaging, and interactive.

---

## ✨ Features

- 📖 Explain cooking concepts in simple language
- 🍳 Real-life cooking examples
- ❓ Generate interactive quizzes
- ✅ Evaluate learner answers with personalized feedback
- 🎓 Complete structured learning sessions
- 💬 Ask cooking-related questions and receive AI-generated responses
- 👩‍🍳 Friendly AI Tutor Persona ("Lavanya")

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ChefMate-AI.git
```

2. Navigate to the project

```bash
cd ChefMate-AI
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Create a `.env` file

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

5. Run the application

```bash
streamlit run app.py
```

---

## ☁️ Deployment

This application can be deployed easily using **Streamlit Community Cloud**.

Store your Gemini API key securely in **Streamlit Secrets** instead of uploading your `.env` file.

Example:

```toml
GOOGLE_API_KEY="YOUR_GEMINI_API_KEY"
```
